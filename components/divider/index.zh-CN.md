区隔内容的分割线。

## 何时使用

- 对不同章节的文本段落进行分割。
- 对行内文字/链接进行分割，例如表格的操作列。

## API

| 参数 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| type | 水平还是垂直类型，可选值为 `horizontal` `vertical` | string | `horizontal` |
| dashed | 是否虚线 | boolean | `false` |
| orientation | 分割线标题的位置，可选值为 `left` `right` `center` | string | `center` |
